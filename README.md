Project Year : 2018
Project name : 사용자 접속 및 로그 관리 프로그램
TARGET : Centos 7(Linux)

--- Description ----
명령어 사용법 : usrlog [-option]
[-option]
    -install : 프로그램 초기화 명령어
    -update  : 계정기록 최신화 처리
    -a       : 계정 정보 및 요약정보 노출
    -u USER_NAME : USER_NAME에 해당하는 유저의 접속 정보 출력
    -u USER_NAME -n NUMBER : 해당되는 번호(NUMBER)의 명령어 사용기록 출력
